# Data Cleaning & Reporting Automation

## Project Overview
This project automates data cleaning and reporting workflows using Python.

## Features
- Removes duplicate records
- Handles missing values
- Standardizes text formatting
- Generates cleaned datasets
- Creates summary reports
- Produces visual analytics automatically

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Output Files
- cleaned_data.xlsx
- summary_report.txt
- data_summary.png

## Outcome
The project improves data quality and reporting efficiency by automating common preprocessing and reporting tasks.
